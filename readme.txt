
Cube main est une application android en realité augmenté, qui permet d'afficher un cube en 2D, puis dés qu'il détecte la main, il transforme le cube de 2D en 3D et le faire tourner dés qu'on clique sur la main

1. pour l'instalation et l'integration du OpenCV :

 * Telchargement d'OpenCV :

       il faut télécharger la version 3.1.0

le lien :

      https://opencv.org/releases.html
